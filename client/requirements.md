## Packages
recharts | For visualizing financial data (revenue vs expenses)
framer-motion | For smooth page transitions and micro-interactions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes safely

## Notes
Authentication is handled by Replit Auth (use-auth hook).
File uploads use FormData to POST /api/files.
Theme should be professional blues and greens.
Fonts: 'Inter' for UI, 'Outfit' for headings.
